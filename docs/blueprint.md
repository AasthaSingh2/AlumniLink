# **App Name**: AlumniLink

## Core Features:

- Alumni Directory: Searchable directory with alumni profiles.
- Profile Page: Detailed profile page with career journey, achievements, and skills. Customizable privacy settings for contact information.
- Events & Webinars: Listing of upcoming reunions, webinars, and workshops with RSVP and automated reminders.
- Mentorship Matching: Connect alumni seeking mentorship with relevant mentors using a matching tool based on skills, industry, and interests. AI recommends potential mentors based on the seeker's profile.
- Opportunities Board: Internship and job postings from alumni companies.
- Donation Progress: Display donation goals and progress bars to encourage contributions. Recognize top donors.
- Admin Announcements: Allow administrators to send announcements to alumni.
- Notification System: Real-time notifications for upcoming events, mentorship requests, and responses.

## Style Guidelines:

- Primary color: Royal blue (#4169E1) to inspire trust.
- Background color: Very light blue (#F0F8FF), nearly white, for a clean look.
- Accent color: Golden yellow (#FFD700) for growth and prestige.
- Body and headline font: 'PT Sans', a humanist sans-serif for modern look with a little warmth.
- Professional icons symbolizing community, networking, mentorship, and growth.
- Modern, minimalistic card-based layout with rounded corners. Intuitive navigation with a top navigation bar for global actions and a bottom navigation bar for primary sections. Admin panel accessible via a sidebar with role-based access control.
- Subtle animations for buttons and transitions to enhance premium feel.
- Responsive design achieved through a flexible grid system and media queries, ensuring optimal viewing experience across devices. Adherence to accessibility standards (WCAG 2.1) with proper contrast ratios, keyboard navigation, and ARIA attributes for screen reader compatibility.
